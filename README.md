## 🚀 简介

**ISPnewmer** 是一个为图像信号处理（Image Signal Processor）工程师设计的可视化小工具，  
用于快速调整 **Gamma、LUT、Noise、Sharpen 等参数曲线**。

## ✨ 功能亮点

- 🎨 **可视化曲线调节**：所见即所得，不用手写配置文件。  
- ⚙️ **实时预览效果**：每次拖动滑块，都能立刻看到变化。  
- 🧘‍♂️ **曲线拟合**：快速调整批量参数值。
